Isotherm Fitting

This module fits adsorption isotherms (e.g., Langmuir, Freundlich) to experimental single-component data.

Isotherm Fitting at Multiple Temperatures

This module fits isotherm models across multiple temperatures and estimates thermodynamic parameters like adsorption enthalpy.

Mixture Isotherm Calculation

This module computes multicomponent adsorption equilibria using Ideal Adsorbed Solution Theory (IAST) or other models.

Breakthrough Simulation

This module simulates breakthrough curves for adsorption columns using input from isotherm data and operational parameters.
